**Plain: Plugin for Predicting the Usability of Mobile User Interfaces**

PLAIN is an automatic evaluation plugin designed to detect defects related to the quality of mobile user interfaces. The plugin measures several metrics that represent state-of-the-art quality attributes commonly used to predict interface quality from a usability perspective.

Given a mobile application as input, the plugin analyzes the interface and generates a list of detected usability defects using quantitative evaluation metrics and predefined defect types.

The plugin was evaluated on four open-source mobile applications. The obtained results confirm that the tool can accurately assess the quality of mobile user interfaces and effectively identify usability issues.
